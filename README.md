# Aura Design Studio — Landing Page

A modern, futuristic landing page for **Aura Design Studio**, built with a dark minimalist aesthetic and neon green accent.

## Preview

> *"We craft digital experiences that blend static precision with fluid motion."*

## Tech Stack

- **HTML5** — Semantic markup
- **Tailwind CSS** (via CDN) — Utility-first styling
- **Google Fonts** — Manrope typeface
- **Material Symbols** — Icon set

## Features

- Dark mode glassmorphism UI
- Responsive design (mobile, tablet, desktop)
- Floating sticky navigation bar
- Split-screen hero section with ambient glow effects
- Services grid with hover interactions
- Selected works portfolio showcase
- Newsletter subscription form
- Oversized footer with call-to-action

## Design Highlights

| Element | Detail |
|---|---|
| Primary Color | `#4cdf20` (Neon Green) |
| Background | `#152111` (Deep Green-Black) |
| Font | Manrope (200–800 weight) |
| Style | Glassmorphism, noise texture overlay |
| Border Radius | Rounded (`2rem` / `3rem`) |

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/XfufuX16/LandingPage---AuraDesignStudio.git
   ```
2. Open `index.html` in your browser — no build step required.
